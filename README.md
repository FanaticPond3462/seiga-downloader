# seiga-downloader
[Seiga Downloader - Chrome ウェブストア](https://chrome.google.com/webstore/detail/dgfdpehcgeilbiifcaajajlcbfkaokfh)

## 概要
ニコニコ静画の作品ページに画像ダウンロード用のリンクを追加します。  
『Seiga Download』の代替になります。
